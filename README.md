# 🚀 Productivity Hub - Neuroscience-Based Tools

A comprehensive web application that combines cutting-edge neuroscience research with practical productivity tools to help you optimize your cognitive performance and achieve peak productivity.

## 🌟 Features

### 🧠 Nootropics Navigator
- **Personalized Recommendations**: Get customized nootropic stacks based on your specific goals
- **Comprehensive Library**: Explore detailed information about 50+ nootropics, supplements, and compounds
- **Scientific Backing**: Each recommendation includes mechanisms of action, considerations, and scientific context
- **Goal-Based Approach**: Choose from Energy & Endurance, Cognitive & Memory Enhancement, or Hormonal Balance

### 🎵 Binaural Beats Generator
- **Brainwave Entrainment**: Harness the power of binaural beats to optimize your mental state
- **Frequency Presets**: Quick access to scientifically-backed frequency ranges:
  - **Delta (0.5-4 Hz)**: Deep sleep and healing
  - **Theta (4-8 Hz)**: Meditation and creativity
  - **Alpha (8-13 Hz)**: Relaxation and focus
  - **Beta (13-30 Hz)**: Alertness and concentration
  - **Gamma (30-100 Hz)**: Insight and peak performance
- **Custom Controls**: Adjust frequency (0.5-40 Hz) and volume to your preference
- **Real-time Generation**: Uses Web Audio API for high-quality audio generation
- **Educational Content**: Learn about brainwave states and optimal usage

## 🎯 How to Use

### Getting Started
1. Open `index.html` in a modern web browser
2. Navigate between tabs using the top navigation bar
3. Each tool is designed to be intuitive and self-explanatory

### Nootropics Section
1. **Select Your Goal**: Choose from three main categories
2. **Review Recommendations**: Get personalized stack suggestions with dosages
3. **Explore the Library**: Dive deep into individual compounds
4. **Read Considerations**: Always review safety information before use

### Binaural Beats
1. **Choose a Preset**: Start with a frequency preset that matches your goal
2. **Use Headphones**: For best results, use stereo headphones
3. **Adjust Settings**: Fine-tune frequency and volume as needed
4. **Start Session**: Begin with 10-15 minute sessions
5. **Find Your Environment**: Use in a quiet, comfortable setting
